Для развертывания сайта необходимо выполнить следующие шаги:
1) установить node js https://nodejs.org/ru, в процессе установить Node Package Manager
2) Установить зависимости, в терминале ввести npm i
3) настроить файл .env
4) ввести npm run start, перейти по ссылке localhost:3000
5) для доступа к административной панели добавить /admin (localhost:3000/admin)
